Existe correlao entre a quantidade de seguidores e o engajamento total dos parlamentares ?

Consideraremos nesta anlise se existe correlao entre as variveis nmero de seguidores e engajamento total dos congressistas. Como j fora discutido em questes passadas sabido que os nmeros diferem muito de um parlamentar para outros em nmeros de seguidores e engajamento. Ento ignorando a discrepncia entre os pontos, podemos perceber que a correlao levando a escala linear tem a seguinte distruibuio.

validos %>%
   ggplot(aes(x = seguidores, y = engaj_total)) + 
  geom_point() +
  labs(title = "Distribuio dos seguidores pelo engajamento total",
             y = "Engajamento", x = "Seguidores")

Considerando a distribuio utilizando uma escala com log

validos %>%
   ggplot(aes(x = seguidores, y = engaj_total)) + 
  geom_point() +
  scale_y_log10() + 
  scale_x_log10() +
  labs(title = "Distribuio dos seguidores pelo engajamento total",
             y = "Engajamento Total", x = "Seguidores")
## Warning: Transformation introduced infinite values in continuous y-axis

Percebemos que visualmente a correlao entre os pontos maior se considerarmos a escalar logaritmca.

Embasando nos coeficientes de correlao linear, Pearson e Spearman, podemos perceber que as variavis tendem a mudar juntas mas em uma taxa no constante.

validos %>%
    summarise(pearson = cor(seguidores, engaj_total, method = "pearson"), 
            spearman = cor(seguidores, engaj_total, method = "spearman"),
            kendall = cor(seguidores, engaj_total, method = "kendall"))
## # A tibble: 1 x 3
##   pearson spearman kendall
##     <dbl>    <dbl>   <dbl>
## 1   0.290    0.628   0.448
validos %>%
    summarise(pearson = cor(log(seguidores), log(engaj_total), method = "pearson"), 
            spearman = cor(log(seguidores), log(engaj_total), method = "spearman"),
            kendall = cor(log(seguidores), log(engaj_total), method = "kendall"))
## # A tibble: 1 x 3
##   pearson spearman kendall
##     <dbl>    <dbl>   <dbl>
## 1     NaN    0.628   0.448

Neste ponto percebemos como os coeficientes de Spearman e Kendall no se alteram em transformaes monotnas, j que continuam com os mesmo valores utilizando log. J nesse caso a distruibuio de Pearson no se aplica.
